Название: Калькулятор ГСМ (Утренний отчёт)
Цель: Автоматизация расчёта расхода топлива для 4 автомобилей при заполнении путевых листов.
Сценарий: Водитель утром вводит данные за прошедшую смену → приложение считает расход → выводит остаток для вписывания в новую путёвку.

1. Интерфейс
Выбор параметров:
Кнопки автомобилей: 181 | 599 | 750 | 883
Кнопки сезона: Лето | Зима
Поля ввода:
Дата (авто: сегодня)
Одометр: выезд (км)
Одометр: заезд (км)
Остаток топлива при выезде (л)
Выдано топлива (л)
Моточасы (часы) — скрыто для 883
Холостой ход (часы) — для всех
Чекбоксы (только для 883):
☑ Груз (+1,00 л/100км)
☑ Городские условия (+0,64 л/100км)
Убрано: Кнопки «Выезд/Заезд», история, экспорт, LocalStora


нормы расходов будут приложены в отдельном файле xlsx
Для 883: моточасы = 0 (нет доп. оборудования). Надбавки: Груз +1,00 л/100км, Город +0,64 л/100км.



## Формулы расчёта

Пробег = Одометр(заезд) − Одометр(выезд)

Норма_итого = База + (Груз ? 1,00 : 0) + (Город ? 0,64 : 0)  // только для 883

Расход_пробег = (Пробег / 100) × Норма_итого
Расход_мото = Моточасы × Норма_М/Ч  // для 883 = 0
Расход_холостой = Холостой_ход × Норма_Х.Х

Общий_расход = Расход_пробег + Расход_мото + Расход_холостой

Остаток_сырой = (Остаток_выезд + Выдано) − Общий_расход

// Округление остатка:
Если (Остаток_сырой - floor(Остаток_сырой)) >= 0,5:
    Остаток_итого = ceil(Остаток_сырой)
Иначе:
    Остаток_итого = floor(Остаток_сырой)
